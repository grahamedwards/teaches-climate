Instructions for exporting slides as a PDF

  1. Click on the "PDF" link on the prior slide (↑)
  2. Open the in-browser print dialog, e.g.
    • ctrl+P
    • cmd+P
    • File→Print
  3. Set Destination to "Save as PDF".
  4. Set the Layout to "Landscape".
  5. Set Margins to "None".
  6. Enable the Background graphics option.
  7. Click "Save"
